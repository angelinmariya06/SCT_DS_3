# SkillCraft Technology - Task 3
## Bank Marketing – Decision Tree Classifier
### 📌 Objective

Build a Decision Tree Classifier to predict whether a customer will purchase a product or service based on their demographic and behavioral data using the Bank Telemarketing dataset.

### 📂 Dataset

- **Bank Telemarketing Dataset**
- Source: Kaggle (based on the UCI Machine Learning Repository)

### 🛠️ Tools & Libraries Used

- Python
- Pandas
- Matplotlib
- Scikit-learn
- Jupyter Notebook

### 📋 Tasks Performed

- Loaded the dataset.
- Explored the dataset using `head()`, `info()`, and `describe()`.
- Checked for missing values.
- Encoded categorical variables using `LabelEncoder`.
- Split the dataset into training and testing sets.
- Built a Decision Tree Classifier.
- Predicted customer purchase decisions.
- Evaluated the model using:
  - Accuracy Score
  - Confusion Matrix
  - Classification Report
- Visualized the Decision Tree.
- Identified the most important features influencing predictions.

### 🤖 Machine Learning Model

- **Algorithm:** Decision Tree Classifier

### 📊 Model Evaluation

The model was evaluated using:

- Accuracy Score
- Confusion Matrix
- Precision
- Recall
- F1-Score

### 📈 Visualizations

- Decision Tree Visualization

### 🔍 Key Findings

- A Decision Tree model was successfully trained to predict customer purchase decisions.
- Categorical variables were converted into numerical values using Label Encoding.
- Feature importance analysis identified the variables that contributed most to the prediction.
- The Decision Tree visualization provided an interpretable representation of the model's decision-making process.
